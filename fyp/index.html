Improve learning resource recommendations for students

Created by Cathal Geoghegan / @_Kotl

Introduction

I have been investigating the use of machine learning algorithms to generate recomendations from an educational corpus. Focusing mainly on LDA, KNN and Word2Vec.

Specifially I have been investigating the temporal performance and the quality of recomendations generated by the above models on a collection of 30,000 papers from arXiv.

Motivation

The internet has proved an invaluable tool in providing access to knowledge. However the volume of data is far too large to be finely catalogued by humans. For this we need to use machines to automate this task.

The overall aim of this is to try and choose a model to use when generating recomendations for students.
